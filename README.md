> 2029年，新浜市。第四次大战的硝烟散尽，人的意识可以化为数据，义体比血肉更常见。
>
> 你是公安9课的草薙素子少佐——全身义体的电子战王牌。案件、政争、潜入、网络深潜，太平洋沿岸这座城市的每一条回线都可能通向真相，或通向你自己的 Ghost。

## ✦ 特色

- **S9 OPS.TERM 作战终端界面**：CRT 荫罩与扫描线、绿磷光与琥珀双色温、POST 开机自检开幕；进入网络潜入层时全机换极性
- **SHELL-DIAG 状态自检面板**：义体负荷、电子脑防壁、热光学迷彩电量、追踪计、线索账本、Ghost 波形（永远校准失败）——每一轮剧情实时更新
- **行动四模式**：发话 / 行动 / 电子战 / 敕令。电子战含表层快骇与深潜共 14 种动词，负荷与追踪自动结算
- **三骰裁定**：作战骰、GHOST 直觉骰、入侵判定骰，五档结果严格裁定成败
- **放饵协议**：回线被分接时，你打的字就是敌人窃听的内容——故意喂假情报，将计就计
- **14 个开局**：从外交官灭口案到傀儡师事件终局，素子视角为主，另有巴特、陀古萨、荒卷乃至フチコマ集体意识视角
- **忠于原作**：剧情严格贴合原作漫画时间线（2029.3–2030.9），人物口癖、译名、政治设定对齐内嵌世界书；日文沙盒模式让 AI 以日文构思再译出中文正文
- **酒馆生态兼容**：SillyTavern 角色卡（PNG/JSON）、世界书高级字段、正则脚本、预设导入

## 怎么玩

游戏本身**不自带 AI**，你接入自己的 AI 接口来驱动世界：

1. 打开在线地址（或用浏览器打开）。
2. 接入你自己的 AI 接口地址与密钥（OpenAI 兼容 / Claude / Gemini 均可）。
3. 选一个开局，开始。

存档、记忆、密钥等一切数据仅存于你的浏览器，不上传、无服务器、无数据库。
